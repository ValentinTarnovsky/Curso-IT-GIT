Git & GitHub

Añadir Email y Nombre en git:

1. git config --global user.email "email@example.com"
2. git config --global user.name "Fullname"

Crear y Subir Repositorio de forma Local:

1. git init
2. git add . | git add [archivo]
3. git commit -m "Nombre"

Conectar Git con Git Hub:

1. generar un Token en GitHub
2. git config --global user.password "token"
3. git config --global credential.helper store

Crear repositorio en GitHub:

1. Crear repositorio desde el perfil
2. copiar el comando de git remote add origin link

Subir el repositorio Local a GitHub:

1. git push -u origin "branch name (Main)"
2. git add .
3. git commit -m "nombre"
4. git pull

Comandos Utiles de GIT:

- git clone "html link" : Clonar un repositorio
- git pull : Traer las actualizaciones del repositorio
- git log --online : ver la lista actualizaciones realizadas
- git status : revisar si tu archivo local esta actualizaco al repositorio
- git restore --staged . : restaurar un add cuando fue enviado a preparacion
- git reset --soft HEAD¬1 : restaura un commit cuando fue enviado (soft para no perder el estado actual)
- git reset --hard HEAD¬1 : restaura un commit cuando fue enviado (hard para borrar el actual volviendo al anterior)
- git clean -fd: limpia los archivos modificados

Que es una Rama ("branch")

Una rama representa una linea independiente de desarrollo,
es como crear un nuevo área de trabajo con su área de ensayo y su historial.
Los nuevos commits se rgistran de forma independiente en el historial de la rama,
lo que se conoce como forks en el historial de proyecto.

Comandos relacionados al Branch

- git branch: Listar las ramas
-git branch nombre-rama: Crear rama con el nombre - git checkout nombre-rama: Para moverse a la rama -git
